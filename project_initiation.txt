1. Tạo 2 thư mục backend, frontend

2. Frontend
    2.1. Khởi tạo
    -   npm create vite@latest .
    -   npm install
    -   npm run dev

    2.2. Cài Tailwind CSS
    -   npm install tailwindcss @tailwindcss/vite

    2.3. Cài daisyUI (giống chakraUI)
    -   npm i -D daisyui@latest

    2.4. Cài thư viện React-Icons
    -   npm install react-icons --save

    2.5. Quản lý điều hướng (routing) trong ứng dụng React
    -   npm i react-router-dom

    2.6. Cài React OAuth2 | Google
    -   npm install @react-oauth/google@latest
    -   tạo .env:
        -   nếu dùng npx create-react-app name thì env phải là REACT_APP_...
        -   nếu dùng npm create vite@latest name thì env pahir là VITE_...
    -   cấu hình main.jsx (nhớ log để check xem env có hợp lệ ko)

    2.7. Cài thư viện call API
    -   npm i axios

    2.8. Cài thư viện animation
    -   npm i framer-motion

    2.9. Cài thư viện thông báo
    -   npm i react-hot-toast

3. Backend
    3.1. Tạo package.json
    -   npm init -y

    3.2. Thêm "type":"module" vào package.json

    3.3. Cài dependencies cần thiết
    -   npm i express nodemon mongoose dotenv jsonwebtoken bcryptjs cookie-parser cloudinary socket.io crypto

    3.4. Cài Google Auth Lib
    -    npm i google-auth-library

    3.5. Cài Gmail SMTP
    -   npm i nodemailer

    3.6. Cài Cors
    -   npm i cors
